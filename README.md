# Keras

Описание
этот проект - это нейросеть, которая умеет различать изображение котов и собак. Моднль обучаеться на изображениях и затем может предсказывать, кто изображен на фотаграфии кот или собака

Технологии
Keras
OpenCV
NumPy
Matplotlib
ImageDataGenerator

Установка и запуск
1.Загружаются изображения котов и собак
2. Данные делятся на train и test
3.Изображения приводятся к размеру 128*128
4.Нейросеть обучается находить признаки
5.На выходе получает: cat или dog
